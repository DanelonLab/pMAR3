# pMAR3
The repository contains supplementary materials for the article "A synthetic cell with integrated DNA self-replication and lipid biosynthesis" by Restrepo et al.
This includes the maps and sequences of the plasmids listed in Table S1 and the source data of the LCMS experiments reported in Figure S8. The SMELDit and pySMELDit codes used for image analysis can be found on the homepage @DanelonLab.
