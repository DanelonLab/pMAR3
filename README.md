# pMAR3
The repository contains supplementary materials for the article "A synthetic cell with integrated DNA self-replication and lipid biosynthesis" by Restrepo et al. published in Nature Communications.

Content: the maps and sequences of the plasmids listed in Table S1, the source data of the LCMS experiments reported in Figure S8, the SMELDit and pySMELDit codes used for image analysis (also available as separate repositories on the homepage @DanelonLab).
